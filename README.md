
# WeChat
---
                                  ![githubLogo](./效果/程序.PNG "Wechat")      
---                          
这是一个网络通信的聊天工具，支持局域网中免流量聊天，发文件，通信内容均采用TSL加密通信，放消息窃听，同时支持私聊和群聊，离线消息发送，
以及群发文件，这是最初始的版本，版本还在更新中
